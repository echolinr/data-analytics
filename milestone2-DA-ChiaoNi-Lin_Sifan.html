1. Product

Briefly describe the data product you intend to build (You can copy this from previous submissions) (5 pts)

We intend to build the app that helps people to get their used-car value estimations or to get used-car price which they are looking for. Basically, a user needs to choose or type make, model, location, year and mileage. Then the app can predict the price range of the given used car so that the user can judge whether the price is reasonable or not.

The application we want to build takes the following inputs:

  • year of make
  • brand
  • model
  • transmission
  • mileage
  • zip code

We will output an estimated price range based on user inputs.

  • lowest value
  • highest value
  • most likely value

In addition to the basic estimation, we plan to use more features to narrow down the estimation range. An advanced function is to ask users to enter detailed information such as vehicle condition and equipment. Therefore, the app is able to return more accurate value to users.

More dimensions we ask users to help narrow down the range can be:

  • engine type
  • vehicle type (limo, economy)
  • repair/damage
